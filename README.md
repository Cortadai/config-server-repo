Este repositorio es un **servidor de configuración centralizado para microservicios Spring Cloud**, que almacena archivos de configuración `.properties` para tres servicios: Employee Service (puerto 8081), Department Service (puerto por defecto) y Organization Service (puerto 8083). Cada servicio está configurado para conectarse a su propia base de datos MySQL (employee_db, department_db y organization_db respectivamente) utilizando Hibernate/JPA con MySQL8Dialect, y todos se registran en un servidor Eureka de descubrimiento de servicios en el puerto 8761. El repositorio sigue el patrón de Spring Cloud Config, permitiendo que las aplicaciones microservicios obtengan su configuración de forma centralizada desde este repositorio Git, facilitando la gestión de configuraciones sin necesidad de redesplegar los servicios cuando cambian parámetros como credenciales de base de datos, URLs de conexión o propiedades personalizadas.
